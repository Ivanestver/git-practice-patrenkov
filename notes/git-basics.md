Что такое Git?
Git - система контроля версий.

Что такое репозиторий?
Хранилище - папка проекта, в которой система Git отслеживает все изменения файлов, хранит историю версий, информацию об авторах и времени правок.

Зачем нужны коммиты?
Для фиксации изменений проекта с целью хранения истории проекта и возможности вернуться к предыдущей стадии.
